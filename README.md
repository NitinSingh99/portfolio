# Nitin Singh - Portfolio Website

A modern, responsive portfolio website built with HTML, Tailwind CSS, and vanilla JavaScript.

## Features

- **Responsive Design**: Optimized for all devices (mobile, tablet, desktop)
- **Dark Mode**: Toggle between light and dark themes
- **Smooth Animations**: Scroll-triggered animations and transitions
- **Modern UI**: Glass morphism effects and gradient backgrounds
- **SEO Optimized**: Semantic HTML5 structure
- **Performance**: Optimized images and minimal JavaScript

## Project Structure

```
portfolio/
├── index.html          # Main HTML file
├── css/
│   └── styles.css      # Custom CSS styles
├── js/
│   └── main.js         # JavaScript functionality
└── README.md          # Project documentation
```

## Sections

1. **Hero Section**: Introduction with animated code card
2. **Stats Section**: Key metrics and achievements
3. **Portfolio**: Recent projects showcase
4. **Experience**: Work history timeline
5. **Testimonials**: Client feedback
6. **Footer**: Contact information and social links

## Technologies Used

- **HTML5**: Semantic markup
- **Tailwind CSS**: Utility-first CSS framework
- **Vanilla JavaScript**: Interactivity and animations
- **Google Fonts**: Typography (Montserrat, Inter, Fira Code)
- **Material Symbols**: Icon library

## Getting Started

1. Clone or download the project
2. Open `index.html` in your browser
3. No build process required - it's a static site

## Customization

### Colors
Edit the CSS variables in `css/styles.css`:
```css
:root {
    --primary: #2563eb;
    --secondary: #3b82f6;
    /* ... */
}
```

### Content
Update the content directly in `index.html`:
- Personal information
- Project details
- Experience timeline
- Testimonials

### Fonts
Modify the Google Fonts links in the HTML head section.
